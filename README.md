Tienda de Accesorios

Proyecto de e-commerce enfocado en el diseño UI y desarrollo frontend, con especial atención a la experiencia visual del usuario, la interacción y la creación de una interfaz atractiva e intuitiva.

✨ Descripción del proyecto

Sitio web de una tienda de accesorios desarrollado desde cero, donde se trabajó la estructura, diseño y comportamiento de una página comercial.

El objetivo principal fue crear una experiencia de compra visualmente agradable mediante una interfaz moderna, incorporando animaciones, transiciones y pequeños detalles interactivos para mejorar la navegación del usuario.

🚀 Características principales

 Página de inicio con diseño atractivo y secciones informativas.
 Catálogo de productos con presentación visual de los artículos.
 Formulario de suscripción por email.
 Diseño responsive adaptable a diferentes dispositivos.
 Animaciones y microinteracciones para mejorar la experiencia de usuario.
 Efectos de scroll con zoom en imágenes.
 Transiciones suaves y efectos hover en elementos interactivos.
 Menú hamburguesa dinámico implementado con JavaScript.
 
🛠️ Tecnologías utilizadas

HTML5

CSS3

Sass

JavaScript

Bootstrap

📌 Objetivos del proyecto

Practicar maquetación frontend y diseño responsive.
Aplicar principios de UI/UX para mejorar la navegación.
Implementar interacciones dinámicas utilizando JavaScript.
Experimentar con animaciones y efectos visuales en páginas web.
